Typescript Demos

Some very basic demos about using typescript and the DOM. Open the browser console to see logging generated by some of the demos.

demo01.html — Writes to the console.

demo02.html — Writes to the console using a formatting function.

demo03.html — Writes to the console using logger.

demo04.html — Logs a message to the console when the DOM is ready to be manipulated.

demo05.html — Updates text content of an element in the DOM.
