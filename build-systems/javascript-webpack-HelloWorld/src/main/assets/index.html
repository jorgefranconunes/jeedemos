Javascript Demos

Some very basic demos about using Javascript and the DOM. Open the browser console to see logging generated by some of the demos.

demo01.html — Writes to the console.

demo02.html — Uses a logger to write to the console.

demo03.html — Uses a listener for the DOMContentLoaded event to be notified when the DOM is ready for manipulation.

demo04.html — Changes the text content of an existing element.

demo05.html — Uses window.setInterval(...) to display the updated current time on the page.
