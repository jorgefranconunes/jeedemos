= Simple sbt multi-project with dependencies

A multi-project


To build:

----
sbt package
----


To run in the development environment:

----
sbt projCli/run
----


To run on the command line after a build:

----
java -jar ./lib/jars/helloworld-cli_2.13-0.1.0-SNAPSHOT.jar
----
